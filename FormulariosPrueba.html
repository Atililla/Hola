ATRIBUTOS PARA LOS INPUT DE TIPO TEXTO

PLACEHOLDER

texto que se muestra en el campo de texto antes de que el usuario escriba


VALUE

valor que se muestra en el campo de texto


SIZE

tamaño del campo de texto


AUTOFOCUS

el campo de texto se selecciona automaticamente al cargar la pagina


REQUIRED

el campo de texto es obligatorio


DISABLED

deshabilita el campo de texto


READONLY

el campo de texto no se puede editar


MAXLENGTH MINLENGTH

longitud maxima y minima del campo de texto


PATTERN

patron que debe cumplir el campo de texto
